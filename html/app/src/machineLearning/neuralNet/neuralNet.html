The Neural Network learns how to generate a color scheme by analyzing the relationships between the colors' red, green and blue values.

A training color scheme is sorted using each of the colors' red poperty, and a value is calculated to indicate on avarage how varied the reds are.

The net has 21 inputs:

  • 1 for the variance in red.
  • 5 x 3 for each of the colors inputs
  • 5 x 1 indicators for each of the colors, that indicate that the color at this index is part of the input.
Only one color acts as an input at a time as per the above indicator.
